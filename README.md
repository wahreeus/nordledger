# NordLedger

Portfolio project focused on designing cloud architecture for a fictional B2B SaaS billing platform.

## Overview

NordLedger is a cloud architecture portfolio project built around a fictional B2B SaaS company that provides subscription billing, invoicing, and analytics services to business clients. The purpose of the project is to demonstrate how an AWS cloud environment could be designed to support security, scalability, reliability, and operational clarity.

Rather than focusing on extensive application feature development, this project emphasizes infrastructure, architecture decisions, deployment planning, and technical documentation. The design process is guided by the AWS Well-Architected Framework and the requirements specification provided in this repository.

## Current Project Status

This project is currently in the early planning and development stages. Figure 1 illustrates a design concept for a client MVP demo that is presently being developed.
