# NordLedger

Portfolio project focused on designing cloud architecture for a fictional B2B SaaS billing platform.

## Overview

NordLedger is a cloud architecture portfolio project built around a fictional B2B SaaS company that provides subscription billing, invoicing, and analytics services to business clients. The purpose of the project is to demonstrate how an AWS cloud environment could be designed to support security, scalability, reliability, and operational clarity.

Rather than focusing on application feature development, this repository focuses on infrastructure, architecture decisions, deployment planning, and supporting technical documentation.

## Scope

This project focuses on:

- high-level cloud architecture
- infrastructure planning
- networking and access design
- deployment approach
- security considerations
- cost and scalability trade-offs
- architecture diagrams and technical documentation

This project does not aim to implement the NordLedger application itself. Instead, it assumes the application already exists and focuses on the cloud environment around it.
